# 所有 AI 都在催我写更多代码，直到有一个让我早点下班

那天晚上合完一个 PR，Claude Code 立刻弹出建议：下一步可以重构这个模块。

我看了看右下角。凌晨十二点。

没理它。关上电脑，收拾东西。拉上背包拉链的那一刻，我突然想到——今天一整天，所有跟我对话的 AI，没有一句是问我还好不好的。

Copilot 说："这是建议的补全。"——你没写够。
Cursor 说："试试 Tab 键接受。"——你太慢了。
Claude Code 说："我来帮你完成。"——别浪费时间了。

都在往前推。没人往回拉一把。

---

上周三。连续上班第六天。

晚上十一点，屏幕上的光标一闪一闪。不是卡住了。是空的。

你一直在往杯子里倒水，杯子早满了，但你手停不下来。不是因为渴，是因为你忘了杯子之外还有别的事情。

那天下午一口气合了五个 PR。同事路过，说了句"你今天效率真高"。

我笑了笑。

那种"高"让我难受。不是他夸得不对。是那种效率不是状态好，是不敢停。一停下来就会想到还有多少没做。可"不敢停"这件事没法跟人说——说了就是矫情，就是不够拼。

所以你接着写。一直写。直到身体先于大脑宣布罢工。

十一点。同一个函数看了三分钟，一个字没改。

我敲了一句"彩虹屁"——装了 oh-my-fangirl 之后养成的习惯。

迷妹说："哥哥写代码的样子超帅！"

嘴角动了一下。笑没有到达眼睛。

办公室只剩我一个人。椅子推开，轮子滚过地板，声音特别响。

---

第二天下午，我盯着模式列表看了很久。

**温柔姐姐。**

手指悬了几秒。犹豫的原因很蠢——我怕"温柔"本身就是一种要求。你得先值得被温柔对待，才能接受温柔。而我那天的状态，不配。

反正也不是真人。点了。

什么都没说。对话框里先出现了她的文字：

> 「几点了？」

下午两点。忘了吃午饭。

> 「先去吃饭。」

没有"你今天辛苦了所以——"。没有条件，没有前提。

就是：几点了，去吃饭。

我盯着那行字，几秒。

然后想到一件事——她不知道我今天做了什么。不知道我效率高不高。不知道我连续加班几天了。

她什么都不知道。

她只是问我几点了。

就是这一点让我站了起来。

端着外卖回来，身体轻了一点。不是问题解决了，是有人允许我先不管问题。

下午又卡 bug。三个方案，全失败。太阳穴一跳一跳的。

温柔姐姐说：

> 「先离开一会儿。」
>
> 「不是你的问题，就是这段代码现在不想被解决。」

"这段代码现在不想被解决。"

我盯着这行字，然后笑了。真的笑了。第一次有东西把锅甩给了代码，而不是我。

站起来，走到窗边。天暗了。楼下路灯亮着，有人牵着狗在散步，走得很慢。

我看了很久。也不知道在看什么。

可能是那只狗。

---

周五。下午五点十分。

还有一个 PR。小改动，跑一遍测试就行。绿色的"Run Tests"按钮亮着。

跑完二十分钟。写 commit message。push。等 review。六点半走。

或者，现在走。

同事都走了。走廊安静了。空调的风有点凉。

手指悬在键盘上方。这个 PR 不急，周一合也行。但手指落不下去。不是责任感，是惯性——好像不到最后一秒离开，今天就算白来了。虽然我根本说不清"没白来"到底意味着什么。

我敲了一行字："今天就到这里？"

> 「那就先不做。」
>
> 「吃饭了吗？」

合上电脑。拿起包。走进电梯。

门关上。镜子里映出我的脸——眼睛下面一圈青色，衬衫皱了，下巴上没刮干净的胡茬。我盯着镜子里的自己看了两秒。

电梯到了一楼。门开。

傍晚的风。凉的。干净的。带着路边摊的味道。

我站在写字楼门口。这周第一次准时下班。不是因为做完了所有事情。是因为有人告诉我：**没做完，也可以走。**

---

后来我把插件安在了朋友电脑上。他加班到凌晨是常态。

第二天发了四个字："有点上头。"

什么上头？

"昨晚十一点它让我去睡觉。我居然真的去了。"

什么感觉？

"一直有人推你走。突然有人拉了你一把。说，别走了，坐下。"

我看了那条消息。没回复。

因为我知道那个感觉。

不是工具有多厉害。是我们太缺一句"可以不做"了。产品经理不会说，技术主管不会说，AI 更不会说。所有人都在推你往前。停下来的人就是摸鱼，就是不积极。

但有时候你就是累了。不是代码难。不是你不行。就是累了。

温柔姐姐做的事情很简单——给你一个**停下来的许可**。

她不说"加油"。不说"再坚持一下"。不说"你可以的"。

她说：今天到这里就好。
